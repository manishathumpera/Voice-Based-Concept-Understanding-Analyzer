Video Demonstration Description

This video demonstrates the Voice-Based Concept Understanding Analyzer (VBCUA) developed using Python, Streamlit, Whisper, and Sentence Transformers.

First, the application is launched, and the user enters the student's name and selects a concept. Then, the student uploads an audio recording explaining the selected concept.

The system analyzes the uploaded audio and displays information such as duration, energy, speech rate, and pause percentage.

Next, the Whisper model converts the speech into text, and the transcript is displayed on the screen.

The system then compares the student's explanation with the reference explanation using semantic similarity analysis and calculates a similarity score.

Based on the audio analysis and semantic similarity, the application generates a final evaluation score.

Finally, a PDF report is generated, the results are saved in the database, and a summary of the student's performance is displayed.

This demonstrates that the VBCUA system can automatically evaluate a student's conceptual understanding through voice input and provide accurate assessment results.
